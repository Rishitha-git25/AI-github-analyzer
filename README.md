🚀 AI Codebase Analyzer

«Turn any GitHub repository into clear, AI-powered insights in seconds.»

---

🧠 What is this?

AI Codebase Analyzer is a full-stack web application that takes a GitHub repository and generates structured insights using AI.

Instead of manually reading code, you get:

- 📌 Summary of the project
- ⚙️ Understanding of structure
- 💡 Key highlights & logic breakdown

---

✨ Features

🔍 Analyze any public GitHub repository
🧠 AI-generated explanation of codebase
📊 Structured output with bullet points
📱 Fully responsive UI (mobile + desktop)
📄 Export results as PDF
⚡ Fast and clean user experience

---

🛠 Tech Stack

Frontend

- React.js
- CSS (Responsive + Custom UI)

Backend

- Node.js
- Express.js

AI Integration

- Groq API (LLaMA-based models)

---

⚙️ How It Works

1. User enters a GitHub repository link
2. Backend fetches repository data
3. Code is processed and sent to AI
4. AI returns structured explanation
5. Frontend displays clean formatted output

🚀 Getting Started

# Clone the repository
git clone https://github.com/your-username/ai-codebase-analyzer.git

# Install dependencies
cd frontend
npm install

cd ../backend
npm install

# Run backend
npm start

# Run frontend
npm start

---

🔐 Environment Variables

Create a ".env" file inside "backend/":

GROQ_API_KEY=your_api_key_here

«⚠️ Never commit your ".env" file»
